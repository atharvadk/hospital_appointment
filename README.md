# hospital_appointment

Project scaffold for the Hospital Appointment system.

This repository has been initialized with formatting and linting tools (black, isort, ruff) and a pre-commit configuration. To get started:

1. Create a virtual environment: python -m venv .venv
2. Activate it: source .venv/bin/activate
3. Install dev dependencies: pip install -r requirements-dev.txt
4. Install pre-commit hooks: pre-commit install

Once set up, use `ruff check .`, `black .`, and `isort .` to lint/format the code.
